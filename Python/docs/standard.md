### 命名规范
<a name= "命名规范" />
所有的命名的第一准则：见名知意

#### 文件名规范
<a name="文件名规范" />
小写(字母、数字),两个单词之间使用`_`连接

eg.
* hello.py
* hello_test.py
* hell1_1.py

#### 类名规范
<a name="类名规范" />
以大写字母开头，两个单词之间使用大写隔开(驼峰式)

eg.
* class Print(object):
* class MyPrint(object):
* class Print1(object):
